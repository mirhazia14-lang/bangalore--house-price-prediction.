# bangalore--house-price-prediction.
****Project Overview**

This project predicts house prices in Bangalore using Machine Learning techniques.
The model analyzes various features such as location, size, number of bedrooms, and square footage to estimate the price of a house.

**Objectives**

- Analyze the Bangalore housing dataset
- Perform data cleaning and preprocessing
- Train machine learning models for price prediction
- Evaluate model performance
  
**Dataset**

The dataset used in this project contains information about houses in Bangalore, including:
- Location
- Size (BHK)
- Total Square Feet
- Number of Bathrooms
- Price
